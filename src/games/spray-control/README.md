# Spray Control Test

เกมนี้เป็นเกมฝึกและทดสอบการคุมเมาส์ตอนยิงสเปรย์ในเกม FPS เช่น Valorant, CS2 หรือ Apex

## วิธีเล่น

1. กดเริ่มเกม
2. หลังนับถอยหลัง ให้วาง crosshair ที่วงเป้า
3. กดคลิกซ้ายค้างเพื่อยิงแบบ full auto หนึ่งแม็ก
4. กระสุนจะมีแรง recoil ดีดขึ้นและแกว่งซ้าย/ขวา
5. ผู้เล่นต้องลากเมาส์สวน recoil เพื่อให้กลุ่มกระสุนอยู่ในวงเป้าให้มากที่สุด

## ทักษะที่วัด

- Spray Control: คุมกลุ่มกระสุนให้อยู่ในเป้า
- Recoil Compensation: ลากเมาส์สวนแรงดีดได้ดีแค่ไหน
- Mouse Control: คุมเมาส์ละเอียดระหว่างยิงต่อเนื่อง
- Aim Consistency: กลุ่มกระสุนกระจุกหรือกระจาย

## การวัดผล

- Hit Rate: กระสุนที่อยู่ในวงเป้ากี่เปอร์เซ็นต์
- Perfect Rate: กระสุนที่อยู่ใกล้กลางเป้ามาก ๆ กี่เปอร์เซ็นต์
- Avg Shot Error: ระยะห่างเฉลี่ยของกระสุนจากจุดกลางเป้า
- Grouping Radius: กลุ่มกระสุนกระจายมากแค่ไหน
- Vertical Control: คุมแรงดีดขึ้นได้ดีแค่ไหน
- Horizontal Control: แก้การแกว่งซ้าย/ขวาได้ดีแค่ไหน

## Result

เกมจะส่งผลกลับผ่าน `onGameComplete(result)` ในรูปแบบ `GameResult` โดยมี `rawData.shots` เก็บข้อมูลกระสุนแต่ละนัด เช่น ตำแหน่งที่ยิง ระยะห่างจากเป้า และว่ายิงโดนหรือไม่
